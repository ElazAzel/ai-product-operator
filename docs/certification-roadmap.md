# Карта сертификаций 2025–2026 — AI Product Operator

## Часть 1. Ландшафт сертификаций

### УРОВЕНЬ 0 — БЕСПЛАТНЫЕ, ДЕЛАТЬ НЕМЕДЛЕННО

**Anthropic Academy** (anthropic.skilljar.com) — бесплатно, 19 курсов, каждый даёт сертификат на LinkedIn.

- Claude 101 — основы: как работают модели Opus/Sonnet/Haiku
- AI Fluency: Framework & Foundations — думать с AI, а не просто вводить запросы
- Building with the Claude API — продуктовый уровень работы с API
- Introduction to MCP / Advanced MCP — подключение инструментов
- Claude Code 101 / Claude Code in Action — агентное кодирование
- AI Fluency для разных аудиторий (преподаватели, студенты, некоммерческие)

**Google AI Essentials** (grow.google/ai-essentials) — бесплатно или ~$150.

**DeepLearning.AI short courses** (deeplearning.ai) — большинство бесплатно.

### УРОВЕНЬ 1 — ПЕРВЫЙ РЕАЛЬНЫЙ ПРОФЕССИОНАЛЬНЫЙ CREDENTIAL

**Claude Certified Architect: Foundations (CCA-F)** — запущен 12 марта 2026.
- 60 вопросов, порог 720/1000
- 5 доменов: агентная архитектура, MCP, Claude Code, cost optimization, production systems
- Сертификат показывает способность проектировать production AI-системы

### УРОВЕНЬ 2 — ВУЗОВСКИЕ И ПЛАТФОРМЕННЫЕ

- Vanderbilt Prompt Engineering Specialization (Coursera) — ~$49/мес
- Google AI Professional Certificate — ~$49/мес
- AWS Certified AI Practitioner (AIF-C01) — $150
- CompTIA AI+ — вендер-нейтральный

### УРОВЕНЬ 3 — ТЕХНИЧЕСКИЕ

- IBM RAG and Agentic AI Professional Certificate (Coursera) — ~$147-245, требует Python

### УРОВЕНЬ 4 — УПРАВЛЕНЧЕСКИЕ

- PMI-CPMAI — $500-800, для ведущих AI-проектов

## Часть 2. Пробелы курса против сертификаций

1. **AI Fluency как системный навык** — нет единого блока
2. **Responsible AI, этика, галлюцинации** — поверхностно
3. **AI Governance и регулирование** — отсутствует
4. **Мультимодальный AI** — не представлен
5. **Cost engineering** — недостаточно
6. **Vector databases** — за кадром
7. **Prompt injection и AI Security** — отсутствует
8. **Certification pathway** — нет карты
